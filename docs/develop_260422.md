1. 비회원으로 시작한 회원이, 다시 어플에 접속 시 동일한 비회원 아이디로 자동으로 로그인 되는지. 아니라면 비회원으로 시작한 회원이 어떻게 다시 본인 계정으로 로그인 할 수 있는지
2. 오일/필터/소모품 패널의 각 항목들의 관리 방식을 저장할 때, 기간 기준 / 날짜 기준 에 따라서 입력해야 하는 필수값이 바뀌어야 함. 기간 기준 > 마지막 교체 날짜 입력 필수 / 마지막 교체 주행거리 선택, 거리 기준 > 마지막 교체 주행거리 필수, 마지막 교체 날짜 선택
3. 차량 등록 시 연료 타입에 LPG 를 추가해야 함. 택시 기사 등 운행 차량 소유자들은 LPG 인 경우가 많음. 전기를 선택했을 때와 유사하게, LPG 선택에 따라 오일/필터/소모품 패널과 충전 패널 등 자동으로 변경되어야 함.
4. 주행거리 분석 등에서 기간 선택 시 최근 30 일/ 기간 지정 뿐 아니라 전체 날짜에 대한 내용도 확인할 수 있도록 선택지 추가
5. 사용자가 처음 앱을 실행할 때, 로그인 속도가 매우 느림. 아마 render 서버가 죽었다가 재기동 되면서 발생하는 이슈로 보임. ping 을 통해 render 서버를 항상 살려놓기로 했는데, 이 부분이 제대로 작동하고 있는지와 개선 계획 세워줘.
6. 이전에 수정했던 주행거리 기록 (기록 보기, 수정, 삭제) 부분이 기존의 기본 정보 패널의 주행 이력 기능과 합쳐졌는지.
